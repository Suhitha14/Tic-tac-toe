# Tic Tac Toe 🎮

A simple **Tic Tac Toe** game built with **HTML, CSS, and JavaScript**.  
Play as **O vs X** on a 3×3 grid. The game detects a winner and allows you to reset and start a new match.

---

## Features 
- 2-player gameplay (Player **O** and Player **X**)
- Winner detection for all possible winning patterns
- Draw detection (when all boxes are filled and no one wins)
- Reset / New Game option
- Clean and responsive UI

---

## How to Play 
1. Player **O** starts first.
2. Players take turns clicking on empty boxes to place their mark.
3. First player to get **3 in a row** wins:
   - Horizontally
   - Vertically
   - Diagonally
4. If all 9 boxes fill and no one wins, it’s a **Draw**.
